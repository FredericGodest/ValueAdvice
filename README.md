# VIA - Value Investing Assistant
VIA est un logiciel assistant les investisseurs.

## Pré-requis
Pour utiliser VIA en local, **Python3** doit être installé.

## Installation
```
pip3 install -r requirements.txt
```

## Utilisation
```
py main.py
```
